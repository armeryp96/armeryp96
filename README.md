## 🚀 Proyectos Destacados

### 1. Infraestructura Empresarial Segura (Windows Server 2022)
Implementación integral de un entorno virtualizado seguro para una organización (`empresa.local`).
- **Servicios Centralizados:** Configuración de AD DS, DNS y DHCP con reservas específicas para clientes Windows y Linux
- **Hardening de Usuarios:** Aplicación de GPOs para restringir el Panel de Control, ocultar unidades de disco y limitar la ejecución de programas no autorizados
- **Seguridad de Red y Host:** Configuración de reglas de Firewall de Windows para tráfico IIS (Puerto 80) y DNS (Puerto 53), además de monitoreo activo con Performance Monitor

### 2. Diseño de Red Empresarial y Segmentación (Cisco)
Diseño y verificación de una red corporativa robusta utilizando equipos Cisco.
- **Segmentación por VLAN:** Creación de subredes aisladas para Administración, Finanzas y Ventas
- **Enrutamiento Inter-VLAN:** Configuración de un router Cisco 2911 para permitir la comunicación controlada entre segmentos
- **Seguridad de Capa 2:** Mitigación de riesgos mediante la desactivación de puertos no utilizados en switches 2960

## 📖 Playbook de Pentesting (Metodología CyberOps)

| Fase (Cyber Kill Chain) | Herramientas | Acción de Auditoría en mi Proyecto |
| :--- | :--- | :--- |
| **Reconocimiento** | `Nmap` | Descubrimiento de servicios activos en las VLANs de Finanzas y Ventas |
| **Análisis de Vulnerabilidades** | `Nessus` / `CVE-DB` | Identificación de debilidades en el servidor IIS y búsqueda de exploits en bases de datos CVE |
| **Explotación** | `Metasploit` | Simulación de compromiso de host para validar si las GPOs detienen la ejecución de payloads |
| **Análisis de Tráfico** | `Wireshark` | Inspección de mensajes ICMP y DNS para detectar anomalías o tráfico de exfiltración |
| **Post-Explotación** | `Infection Monkey` | Simulación de movimiento lateral entre subredes para probar las ACLs del router |
